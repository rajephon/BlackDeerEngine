# BlackDeerEngine
XML script engine for Unity3D

디어화이트 프로젝트의 일부입니다. __아직 개발이 진행 중입니다.__

XML 스크립트를 이용하여 편하게 드라마틱한 동작을 만들거나 게임의 이야기를 연출할 수 있습니다.
